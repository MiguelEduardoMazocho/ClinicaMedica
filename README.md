Descrição

Este projeto consiste em um sistema web para gestão de clínicas médicas, desenvolvido com o objetivo de facilitar a organização de atendimentos, o controle de pacientes e o gerenciamento das rotinas administrativas. A aplicação foi pensada para ser simples, funcional e intuitiva, atendendo às necessidades básicas de uma clínica.

Demonstração

Projeto desenvolvido para fins educacionais e de portfólio. Pode ser executado localmente seguindo as instruções abaixo.

Funcionalidades

Pacientes

- Cadastro de novos pacientes  
- Edição e atualização de dados  
- Listagem de pacientes cadastrados  
- Organização das informações de forma clara  

Consultas

- Agendamento de consultas  
- Visualização de horários marcados  
- Controle de atendimentos realizados  
- Associação entre pacientes e consultas

3. Acesse a pasta do projeto:

5. Abra o arquivo `index.html` no navegador:
    Clique duas vezes no arquivo
     ou
